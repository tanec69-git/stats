Revision Exercise 1: Relations & Functions

Question 3 (b)

Determine the functions $fg$ and $gf$ for the following:

  • $f(x) = \frac{3}{2-x} , \quad x \ne 2$
  • $g(x) = x - 1$

1. Find the composite function $fg$

Logic: $fg(x)$ means $f(g(x))$. Substitute the entire function $g(x)$ into $f(x)$ wherever there is an $x$.
$$fg(x) = f(x - 1)$$
Substitute $(x - 1)$ into the denominator of $f(x)$:
$$fg(x) = \frac{3}{2 - (x - 1)}$$
Expand the bracket and simplify the denominator:
$$fg(x) = \frac{3}{2 - x + 1}$$
$$fg(x) = \frac{3}{3 - x}$$
Determine the Domain: For $f(g(x))$ to be valid, the inner function's output must not equal $2$ (since $x \ne 2$ is the restriction for $f$). Therefore, $g(x) \ne 2 \implies x - 1 \ne 2 \implies x \ne 3$.
Answer: $fg(x) = \frac{3}{3 - x}, \quad x \ne 3$

2. Find the composite function $gf$

Logic: $gf(x)$ means $g(f(x))$. Substitute the entire function $f(x)$ into $g(x)$ wherever there is an $x$.
$$gf(x) = g\left(\frac{3}{2 - x}\right)$$
Substitute $\frac{3}{2 - x}$ into $g(x)$:
$$gf(x) = \left(\frac{3}{2 - x}\right) - 1$$
To combine these terms, convert $1$ into a fraction with a common denominator of $(2 - x)$:
$$gf(x) = \frac{3}{2 - x} - \frac{2 - x}{2 - x}$$
Combine the numerators over the common denominator. Be careful to distribute the negative sign:
$$gf(x) = \frac{3 - (2 - x)}{2 - x}$$
$$gf(x) = \frac{3 - 2 + x}{2 - x}$$
$$gf(x) = \frac{x + 1}{2 - x}$$
Determine the Domain: For $g(f(x))$ to be evaluated, the input $x$ must first pass through $f(x)$. The original restriction for $f(x)$ is $x \ne 2$. The outer function $g(x)$ has no further domain restrictions. Therefore, the restriction remains $x \ne 2$.
Answer: $gf(x) = \frac{x + 1}{2 - x}, \quad x \ne 2$
