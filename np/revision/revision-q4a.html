Solutions for Question 4

$$ f(x) = \begin{cases} -x + 1 & \text{when } -1 \le x < 1, x \in \mathbb{R} \\ 2 & \text{when } x = 1 \\ x^2 & \text{when } x > 1, x \in \mathbb{R} \end{cases} $$

Part (a): Sketch and Function Type

Sketch of the Graph:

Classification:

The function is a many-one function.

Reasoning:

Using the horizontal line test (indicated by the red dashed line at $y = 2$), we can see that a horizontal line intersects the graph at more than one point. Alternatively, we can prove this algebraically by evaluating the function at specific coordinates:

  • At $x = -1$: $f(-1) = -(-1) + 1 = 2$
  • At $x = 1$: $f(1) = 2$

Because $f(-1) = f(1) = 2$ despite $-1 \neq 1$, two distinct inputs map to the exact same output. Therefore, it is a many-one function.

Part (i): Evaluate $f(0)$, $f(1)$, and $ff(2)$

  • Evaluate $f(0)$:
    Since $0$ falls in the interval $-1 \le x < 1$, we use the first piece: $-x + 1$.
    $$ f(0) = -0 + 1 = 1 $$

  • Evaluate $f(1)$:
    Since $x = 1$, we use the second piece.
    $$ f(1) = 2 $$

  • Evaluate $ff(2)$:
    First, evaluate the inner function $f(2)$. Because $2 > 1$, use the third piece: $x^2$.
    $$ f(2) = 2^2 = 4 $$
    Now substitute this back in to find $f(4)$. Because $4 > 1$, use the third piece again.
    $$ f(4) = 4^2 = 16 $$
    Therefore, $ff(2) = 16$.

Part (ii): Find $x$ if $f(x) = 36$

We must determine which piece of the function could yield a value of $36$:

  • For $-1 \le x < 1$, the range is $0 < f(x) \le 2$.
  • For $x = 1$, the output is exactly $2$.
  • For $x > 1$, the function is $x^2$. This is the only piece capable of reaching $36$.

Set up the equation:

$$ x^2 = 36 $$ $$ x = 6 \quad \text{or} \quad x = -6 $$

Since this piece of the function is only valid for $x > 1$, we reject $-6$.

Therefore, $x = 6$.

Part (iii): State the domain and range

Domain:

The domain is the union of the intervals defined for $x$: $[-1, 1) \cup \{1\} \cup (1, \infty)$. Combining these covers all real numbers starting from $-1$.

$$ \{ x \in \mathbb{R} \mid x \ge -1 \} $$

Range:

Looking at the outputs (y-values) for each interval:

  • Piece 1 creates y-values in $(0, 2]$.
  • Piece 2 gives $\{2\}$.
  • Piece 3 gives $(1, \infty)$.

The union of these sets is $(0, \infty)$. Therefore, the range is all real numbers strictly greater than 0.

$$ \{ f(x) \in \mathbb{R} \mid f(x) > 0 \} $$

Part (iv): Inverse Function

Answer: No, there is no inverse function for $f(x)$.

Reason: As established in part (a), $f(x)$ is a many-one function (for example, $f(-1) = 2$ and $f(1) = 2$). For a function to have an inverse, it must be strictly one-to-one (injective) so that every output can be uniquely mapped back to a single input. Because multiple x-values map to the same y-value, the inverse mapping is ambiguous.
