📈 Set Theory Extremes Solution

Question (c):
If $n(A) = 15$ and $n(B) = 10$, what are the greatest and the least values of $n(A \cup B)$?
Use Venn diagrams to illustrate the cases in which $n(A \cup B)$ has these extreme values.

The general formula for the union of two sets is:
$$n(A \cup B) = n(A) + n(B) - n(A \cap B)$$

Since $n(A) = 15$ and $n(B) = 10$ are fixed, the value of $n(A \cup B)$ depends entirely on the intersection $n(A \cap B)$.

1. The Greatest Value of $n(A \cup B)$

To maximize $n(A \cup B)$, we must minimize the intersection $n(A \cap B)$. The smallest possible overlap is $0$. This occurs when sets A and B are mutually exclusive (disjoint sets).

$$n(A \cap B) = 0$$ $$n(A \cup B) = 15 + 10 - 0 = \mathbf{25}$$

Greatest Value: 25

Venn Diagram Illustration: Disjoint Sets

2. The Least Value of $n(A \cup B)$

To minimize $n(A \cup B)$, we must maximize the intersection $n(A \cap B)$. The largest possible overlap occurs when the smaller set is entirely contained within the larger set. Since $n(B) < n(A)$, Set B is a proper subset of Set A ($B \subset A$).

$$n(A \cap B) = n(B) = 10$$ $$n(A \cup B) = 15 + 10 - 10 = \mathbf{15}$$

Least Value: 15

Venn Diagram Illustration: Proper Subset ($B \subset A$)

Notice that inside Set A, there are $10$ items belonging to B, and $5$ items belonging exclusively to A, totaling $15$.
