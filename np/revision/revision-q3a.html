Revision Exercise 1: Relations & Functions

Question 3 (a)

A relation defined from $\mathbb{Z}$ to $\mathbb{Z}$ as:

$g(x) = \begin{cases} x - 1 & \text{if } x \ge 0, x \in \mathbb{Z} \\ x & \text{if } x < 0, x \in \mathbb{Z} \end{cases}$

(i) Find $g(-3), g(2), g(10), g(0), gg(5)$

Logic: Check the condition for $x$ before applying the respective rule.
  • For $g(-3)$: Since $-3 < 0$, apply $g(x) = x \Rightarrow g(-3) = -3$
  • For $g(2)$: Since $2 \ge 0$, apply $g(x) = x - 1 \Rightarrow g(2) = 2 - 1 = 1$
  • For $g(10)$: Since $10 \ge 0$, apply $g(x) = x - 1 \Rightarrow g(10) = 10 - 1 = 9$
  • For $g(0)$: Since $0 \ge 0$, apply $g(x) = x - 1 \Rightarrow g(0) = 0 - 1 = -1$
  • For $gg(5)$: First find $g(5)$. Since $5 \ge 0$, $g(5) = 5 - 1 = 4$. Then find $g(4)$. Since $4 \ge 0$, $g(4) = 4 - 1 = 3$.
Answer: $g(-3) = -3, \quad g(2) = 1, \quad g(10) = 9, \quad g(0) = -1, \quad gg(5) = 3$

(ii) State the domain of g.

Logic: The problem explicitly states the relation is defined "from $\mathbb{Z}$ to $\mathbb{Z}$". Furthermore, the piecewise conditions ($x \ge 0$ and $x < 0$) cover all possible integers.
Answer: The domain is the set of all integers, $\mathbb{Z}$.

(iii) Sketch the graph of $g(x)$.

Logic: Because the domain is restricted to integers ($\mathbb{Z}$), the graph consists of discrete points, not a continuous line.
Calculate a few coordinates to plot:
  • $x < 0$: $(-3, -3), (-2, -2), (-1, -1)$
  • $x \ge 0$: $(0, -1), (1, 0), (2, 1), (3, 2)$

(iv) State the range of g.

Logic: Look at the outputs mapped by the function.
For $x < 0$, the outputs are $\{-1, -2, -3, \dots\}$
For $x \ge 0$, the outputs are $\{-1, 0, 1, 2, 3, \dots\}$
Combining these two sets covers every integer exactly once, except for $-1$, which is produced twice.
Answer: The range is the set of all integers, $\mathbb{Z}$.

(v) Is the above relation a one-to-one or many-to-one function? Explain.

Logic: A function is many-to-one if two or more distinct inputs map to the exact same output.
Looking at the evaluations from our graph/calculations:
  • Evaluate at $x = -1$: Since $-1 < 0$, $g(-1) = -1$
  • Evaluate at $x = 0$: Since $0 \ge 0$, $g(0) = 0 - 1 = -1$
Since $g(-1) = g(0) = -1$, multiple inputs share the same output.
Answer: It is a many-to-one function.
