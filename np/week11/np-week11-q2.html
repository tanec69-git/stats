Probability Distribution Analysis

2. Check whether the following function \( f(x) \) can be defined as the probability distribution of an appropriate random variable. (Use Rules of a Discrete Probability Distribution)

$$ f(x) = \frac{x + 2}{12} \quad \text{for } x = 1, 2 \text{ and } 3 $$

Solution:

For a function to be a valid discrete probability distribution, it must satisfy two fundamental rules:

  1. The probability of each outcome must be non-negative: \( f(x) \ge 0 \) for all \( x \).
  2. The sum of all probabilities over the entire domain must equal 1: \( \sum f(x) = 1 \).

First, let's calculate the values of \( f(x) \) for each value in the given domain (\( x = 1, 2, 3 \)):

  • For \( x = 1 \): \( \quad f(1) = \frac{1 + 2}{12} = \frac{3}{12} \)
  • For \( x = 2 \): \( \quad f(2) = \frac{2 + 2}{12} = \frac{4}{12} \)
  • For \( x = 3 \): \( \quad f(3) = \frac{3 + 2}{12} = \frac{5}{12} \)

Checking Rule 1: Non-negativity

We check if \( f(x) \ge 0 \) for all \( x \):

\( \frac{3}{12} \ge 0 \), \( \frac{4}{12} \ge 0 \), and \( \frac{5}{12} \ge 0 \).

Since all individual probabilities are greater than zero, Rule 1 is satisfied.

Checking Rule 2: Sum of Probabilities

We sum the calculated probabilities to see if they equal 1:

$$ \sum_{x=1}^{3} f(x) = f(1) + f(2) + f(3) $$

$$ \sum_{x=1}^{3} f(x) = \frac{3}{12} + \frac{4}{12} + \frac{5}{12} $$

$$ \sum_{x=1}^{3} f(x) = \frac{3 + 4 + 5}{12} $$

$$ \sum_{x=1}^{3} f(x) = \frac{12}{12} = 1 $$

Since the total sum is exactly 1, Rule 2 is satisfied.

Conclusion: Yes, because both rules are fully satisfied, the function \( f(x) \) can be defined as a valid probability distribution.
