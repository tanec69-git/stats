Range in Set-Builder Notation

Represent the range of the function \(h\) using the set-builder notation.

\[ h(x)=2x-7 \qquad \text{for } 4 \le x < 6, \; x \in \mathbb{R} \]
Step 1: Find the minimum value of \(h(x)\). \[ h(4)=2(4)-7 \] \[ =8-7 \] \[ =1 \]
Step 2: Find the maximum value of \(h(x)\). Since \(x<6\), \[ h(x)<2(6)-7 \] \[ h(x)<12-7 \] \[ h(x)<5 \]
Step 3: Hence, \[ 1 \le h(x) < 5 \]
\[ \text{Range of } h(x) = \{x : 1 \le x < 5,\; x \in \mathbb{R}\} \]
