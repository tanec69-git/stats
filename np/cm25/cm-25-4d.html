🎂 Cake Arrangement Problem 🍓

Learn permutations with identical objects!
A cake shop displays 8 cakes in a row in a display case:
🍫 3 identical chocolate cakes
🧁 3 identical butter cakes
🍓 2 identical strawberry cakes

How many distinct ways can the cakes be arranged:

(i) if there is no restriction

(ii) if the 2 strawberry cakes must NOT be next to each other?

🍫🧁🍓 (i) No Restriction

We have 8 cakes in total.

Since some cakes are identical, use the formula: \[ \frac{8!}{3!3!2!} \] Calculate: \[ = \frac{40320}{(6)(6)(2)} \] \[ = \frac{40320}{72} \] \[ = 560 \]

✅ Answer: \[ \boxed{560} \]

🚫🍓🍓 (ii) Strawberry Cakes NOT Together

First find the total number of arrangements: \[ \frac{8!}{3!3!2!} = 560 \]
Next, find arrangements where the 2 strawberry cakes ARE together.

Treat 🍓🍓 as one object.

We now have: \[ 7 \text{ objects} \] consisting of:
• 🍓🍓 block
• 3 🍫 cakes
• 3 🧁 cakes

Number of arrangements: \[ \frac{7!}{3!3!} \] \[ = \frac{5040}{36} \] \[ = 140 \]
Therefore, arrangements where strawberries are NOT together: \[ 560 - 140 \] \[ = 420 \]
✅ Answer: \[ \boxed{420} \]
