✈️📊 Ticket Price Statistics

Using mathematical notation for transformations of data
In June, all SIA plane ticket prices from Singapore to Japan for November and December are released.

The mean ticket price for all seats is \$1050 with a standard deviation of \$250.

(i) If the airline offers a discount of 15% to all tickets, calculate the new mean and new standard deviation.

(ii) If the airline increases the listed prices by \$200, calculate the new mean and new standard deviation.

(iii) If all ticket prices were doubled, calculate the new mean and new standard deviation.

💸 (i) 15% Discount on All Tickets

\[ \text{Given:} \qquad \text{Mean, } \mu_x = 1050 \] \[ \therefore \text{ Mean, } \mu_y = 0.85\mu_x \] \[ = 0.85 \times 1050 \] \[ = \$892.50 \]
\[ \text{Given:} \qquad \text{Std Dev, } \sigma_x = 250 \] \[ \therefore \text{ Std Dev, } \sigma_y = 0.85 \times \sigma_x \] \[ = 0.85 \times 250 \] \[ = \$212.50 \]
✅ New Mean: \[ \boxed{\$892.50} \] ✅ New Standard Deviation: \[ \boxed{\$212.50} \]

📈 (ii) Increase All Prices by \$200

\[ \text{Given:} \qquad \text{Mean, } \mu_x = 1050 \] \[ \therefore \text{ Mean, } \mu_y = 1050 + 200 \] \[ = \$1250 \]
\[ \text{Given:} \qquad \text{Std Dev, } \sigma_x = 250 \] \[ \therefore \text{ Std Dev, } \sigma_y = \sigma_x \text{ (remain unchanged)} \] \[ = \$250 \]
✅ New Mean: \[ \boxed{\$1250} \] ✅ New Standard Deviation: \[ \boxed{\$250} \]

✖️2️⃣ (iii) Prices are Doubled

\[ \text{Given:} \qquad \text{Mean, } \mu_x = 1050 \] \[ \therefore \text{ Mean, } \mu_y = 2\mu_x \] \[ = 2 \times 1050 \] \[ = \$2100 \]
\[ \text{Given:} \qquad \text{Std Dev, } \sigma_x = 250 \] \[ \therefore \text{ Std Dev, } \sigma_y = 2\sigma_x \] \[ = 2 \times 250 \] \[ = \$500 \]
✅ New Mean: \[ \boxed{\$2100} \] ✅ New Standard Deviation: \[ \boxed{\$500} \]
