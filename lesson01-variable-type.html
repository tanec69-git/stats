Variable Types Practice App

Classify each variable as one of the following:
  • Qualitative
  • Quantitative (Discrete)
  • Quantitative (Continuous)
Helpful Tips:

• Qualitative → categories, labels, names, colours, codes, identifiers.

• Quantitative (Discrete) → counted values, usually whole numbers.

• Quantitative (Continuous) → measured values that can contain decimals.
